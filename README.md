Chatty App
=====================

A simple, multi-user, real-time chat app built with React and JSX.

![Choose a name and start chatting away.](https://github.com/MattLatimer/chatty-app/blob/master/chatty-app.png?raw=true)

### Usage

Install the dependencies and start the servers.

/chatty-app
```
npm install
npm start
```
/chatty-app/chatty-server
```
npm install
npm start
```

open http://localhost:3000

### Dependencies

#### Chatty App
 * babel-core
 * babel-loader
 * babel-preset-es2015
 * babel-preset-react
 * css-loader
 * node-sass
 * sass-loader
 * sockjs-client
 * style-loader
 * webpack
 * webpack-dev-server
 * react
 * react-dom

#### Chatty Server
 * express
 * ws
 * uuid
